# Calculator
# 🧮 Calculator Web Application

A modern and interactive calculator built using HTML, CSS, and JavaScript. This project provides a clean user interface and supports basic arithmetic operations with real-time calculation functionality.

## 🚀 Features

* Addition, Subtraction, Multiplication, and Division
* Percentage (%) calculations
* Parentheses support for complex expressions
* Decimal number calculations
* Clear (C) button to reset the display
* Responsive and user-friendly interface
* Dynamic expression evaluation using JavaScript

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript

## 📁 Project Structure

calculator/
│

## 📋 How It Works

1. Click the calculator buttons to enter numbers and operators.
2. Build a mathematical expression.
3. Press "=" to calculate the result.
4. Press "C" to clear the display and start over.

## 💡 Supported Operations

| Operation      | Symbol |
| -------------- | ------ |
| Addition       | +      |
| Subtraction    | -      |
| Multiplication | ×      |
| Division       | /      |
| Percentage     | %      |
| Parentheses    | ( )    |
| Decimal        | .      |
